# 自定义视觉界面

# 介绍

GtHawkeye 基于 c++ 实现，完全前后端分离，保证用户使用单一头文件，能快速实现自己的视觉界面。

不需实现算法，不需配置流程，丢掉一切视觉配置的繁琐操作，专注于视觉界面的实现。

可以提供相关动态链接库，可做跨平台，支持 Win/Linux 平台。

# 你是否需要？

是否想要把视觉快速集成到自己的控制软件内？

是否想要针对特定行业定制专属界面？

如果存在以上需求，即可采用这种开发方式。

# 先验知识

建议在开发之前，用户了解 c++ 相关知识。
